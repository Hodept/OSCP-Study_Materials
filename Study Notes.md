# OSCP Study Notes

## The Penetration Testing Lifecycle

##### Lifecycle phases:

A typical penetration test comprises the following stages:

- Defining the Scope
- Information Gathering
- Vulnerability Detection
- Initial Foothold
- Privilege Escalation
- Lateral Movement
- Reporting/Analysis
